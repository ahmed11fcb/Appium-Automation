**Project Overview**

This project demonstrates mobile automation testing on the SwagLabs E-Commerce application using Appium, Selenium, and TestNG. It covers essential mobile gestures and UI interactions to validate app functionality and improve testing efficiency.

**Features / Gestures Implemented**

The automation scripts cover the following gestures and actions:

Scrolling – scroll up/down to locate elements

Swiping – swipe left/right for carousel or menu interactions

Rotation – test app behavior in portrait and landscape mode

Clicking / Tapping – interact with buttons, products, and menus

**Frameworks & Tools Used**

Appium – for mobile automation

Selenium WebDriver – for cross-platform UI interactions

TestNG – for test management, execution, and reporting

**Tested Scenarios**

Login and logout functionality

Adding products to the cart

Navigating through product pages using gestures

Validating UI behavior during rotation and scrolling

**How to Run** 

Clone the repository:  git clone https://github.com/ahmed11fcb/Appium-Automation.git

Open the project in IDE (IntelliJ / Eclipse)

Set up the required mobile emulator or real device

Run tests using TestNG XML file
