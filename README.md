**Project Overview**

--> This project demonstrates mobile automation testing on the SwagLabs E-Commerce application using Appium, Selenium, and TestNG. It covers essential mobile gestures and UI     interactions to validate app functionality and improve testing efficiency.

**Features / Gestures Implemented**

1- The automation scripts cover the following gestures and actions:

2- Scrolling – scroll up/down to locate elements

3- Swiping – swipe left/right for carousel or menu interactions

4- Rotation – test app behavior in portrait and landscape mode

5- Clicking / Tapping – interact with buttons, products, and menus

**Frameworks & Tools Used**

1- Appium – for mobile automation

2- Selenium WebDriver – for cross-platform UI interactions

3- TestNG – for test management, execution, and reporting

**Tested Scenarios**

1- Login and logout functionality

2- Adding products to the cart

3- Navigating through product pages using gestures

4- Validating UI behavior during rotation and scrolling

**How to Run** 

1- Clone the repository:  git clone https://github.com/ahmed11fcb/Appium-Automation.git

2- Open the project in IDE (IntelliJ / Eclipse)

3- Set up the required mobile emulator or real device

4- Run tests using TestNG XML file
